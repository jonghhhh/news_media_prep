언론고시 준비 매뉴얼 — 통합 안내 & 공통 첨삭 기준

초보 준비생을 위한 유형별 실전 매뉴얼 7종입니다. 각 매뉴얼은 ① 기초 강의(개념→준비→구성→작성 원칙) ② 실제 수집 데이터에서 뽑은 우수 사례 심층 분석 ③ 준비생이 초안을 가져오면 바로 봐줄 수 있는 핵심 평가·첨삭 기준과 첨삭 프로세스로 구성됩니다.

근거 데이터: 공개 웹에서 수집·정제한 351건(기존 벡터DB 21 + 신규 크롤 330) 원문. 각 매뉴얼의 사례는 [Dxxxx] id로 dataset/typed_dataset.csv의 원문과 연결됩니다.

매뉴얼 목록 (전형 단계 + 사전 준비)

# 매뉴얼 전형 단계 근거 코퍼스 핵심 사례
01 자기소개서 서류 자소서 34건 합격/불합격 대조 6쌍
02 논술 필기 논술 113건 합격 논술문 6편 해설
03 작문 필기 작문 59건 합격 작문 6편 해설
04 PD 기획안 필기(PD) 기획안 10건 실물 기획안 20여 편
05 현장취재·기사작성 실무 취재·기사 25건 리드·취재 사례 16편
06 면접 면접 면접 37건 면접 후기·질문 26편
07 기초지식 쌓기 상시 토대 기초·시사·직무 215건 시사 이슈맵·추천도서
08 논증구조(Toulmin 중심) 전 유형 논리 뼈대 toulmin 논증의 6요소 기반 사설 6요소 분해·유형별 적용
09 독창적 사고·글쓰기 관점 독창성·차별화(독립) 창의성 이론 13종 기반 이론(발상/분석/평가)+실전 7기법·독창성 첨삭 루브릭

추천 학습 순서: 07 기초지식(토대) → 01 자소서(서류) → 02 논술·03 작문·04 기획안(필기) → 05 현장취재(실무) → 06 면접 → 08 툴민·09 독창적 사고(전 유형 심화). 단, 07은 전 과정 내내 병행. 08 툴민은 02 논술과 함께 읽으면 논증 구조 감각이 상승하며, 전 유형의 논리 뼈대를 잡아주는 '메타 매뉴얼'입니다. 09 독창적 사고·글쓰기는 관점 독창성·차별화를 담당하는 독립 매뉴얼로, 08과 함께 전 유형의 사고 깊이를 더합니다.

전 유형 공통 첨삭 철학 (5축)

모든 매뉴얼의 평가·첨삭 기준은 아래 5개 축을 공통 뼈대로, 유형 특성에 맞게 배점·세부기준을 조정합니다. (논술은 각 20점 100점 만점의 표준 rubric을 그대로 사용 — 앱 채점 기준과 동일)

공통 축 무엇을 보는가 유형별 변주(예)
① 내용·논지 메시지가 분명하고 설득력 있는가 논술=논지 명확성 / 작문=통찰 / 자소서=직무적합·진정성 / 기획안=기획의도 / 기사=사실성 / 면접=직업관
② 구조·구성 흐름이 완결되고 배치가 전략적인가 논술=논리 구조 / 작문=리드·여운 / 기사=역피라미드 / 자소서=두괄식
③ 독창성·관점 남과 다른가, 신선한가 논술=관점 비틀기 / 작문=소재 신선함 / 기획안=차별화
④ 근거·구체성 디테일·사례·팩트로 뒷받침되는가 논술=자료 차별화(책·논문) / 자소서=경험의 구체성 / 기사=취재 밀도
⑤ 표현·문장 읽기 쉽고 문장력이 있는가 전 유형 공통(간결·능동·리듬·톤)

첨삭 공통 프로세스 (준비생이 초안을 가져왔을 때)

  1. 진단(전체 인상): 유형 요건 충족 여부 + 5축 중 가장 약한 축 1~2개 지목
  2. 항목별 지적: 축별로 "무엇이·왜 문제인지" 문장/단락 단위로 근거 제시
  3. 수정 방향 제시: 감점 신호 → 가점 신호로 바꾸는 구체적 처방
  4. 개선 예시(Before→After): 같은 소재로 나쁜 문장→좋은 문장 실연
  5. 재작성 과제 + 자가진단 체크리스트 부여

각 매뉴얼 8~9장에 유형별 배점표 · 감점/가점 신호 · Before→After 예시가 상세히 들어 있습니다.

사용법

  • 강의 교재로: 각 매뉴얼 1~7장(개념~작성 원칙)을 순서대로 학습.
  • 첨삭 기준으로: 각 매뉴얼 8~9장(평가 기준+첨삭 프로세스)을 초안 진단 체크리스트로 사용.
  • 사례 열람: 본문의 [Dxxxx]를 dataset/typed_dataset.csv에서 검색하면 인용 원문 전체를 볼 수 있음.
  • AI 첨삭기 연동(다음 단계): 이 매뉴얼의 평가축·기준을 시스템 프롬프트/RAG 지식으로 넣으면 자동 첨삭 품질을 높일 수 있음.

데이터 한계 (보강이 필요한 영역)

수집 코퍼스가 논술·작문·후기·직무 읽을거리에 편중되어, 면접·실무전형·기출문제 원문이 상대적으로 얇습니다. 해당 매뉴얼(05·06)은 코퍼스 근거와 언론 실무 표준을 구분 표기해 신뢰도를 유지했습니다. 아랑 카페(기출 복원방·후기방, 로그인 필요) 등에서 추가 수집 시 더 두터워집니다.
